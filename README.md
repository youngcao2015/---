# SearchRecordList

##概述

  本例简单实现搜索记录列表，可以自动换行。

##实现
1.数据部分：实现数据去重，最近搜索数据排在最前面，规定搜索记录个数；<br>
2.视图部分：使用搜索数据来填充摆放的标签，当标签不够摆放时，换行继续摆放。
